I like the ofb site, and the fetishes within. However, I found it lacking in usability, and has not appeared to be maintained in some time. There are different ways I would like to browse the stories. An example of a good presentation is AO3. This site also attempts to convert and provide a consistent, desired text format for reading (be it pdf, text, epub)

This site temorarily scans the original to pull new uploads, and presents them in an alternative manner

Contact

The site code is free (as in speech) available on GitHub. Please submit bug reports, feature requests, and feedback there. You can also use the code to host your own instance of this site.

Accessibility

This site aims to follow accessibility standards. It does so by making use of aria accessiblity tags, and by using a contrasting color scheme.

License

Code for this website is available freely at GitHub.

Stories uploaded to the original site were not under any obvious copyright or license. It is a shame, however, for the site owner to have ownership of user-submitted content. Therefore, this site proposes to have stories available under CC-BY-SA
